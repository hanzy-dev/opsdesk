# CV, LinkedIn, dan Portfolio Bullets

Dokumen ini berisi beberapa versi bullet yang siap dipakai ulang. Pilih versi yang paling sesuai dengan ruang yang tersedia.

## Versi Sangat Singkat

- Membangun OpsDesk, aplikasi helpdesk internal berbasis cloud dengan React, Go, AWS Lambda, API Gateway, DynamoDB, Cognito, dan S3.
- Mengimplementasikan autentikasi Cognito, RBAC, workflow tiket, assignment operator, audit trail, dan dokumentasi API interaktif.
- Men-deploy aplikasi secara nyata di Vercel dan AWS SAM dengan arsitektur serverless yang terintegrasi.

## Versi Menengah

- Mengembangkan OpsDesk, aplikasi helpdesk dan ticketing internal berbasis cloud, menggunakan React + Vite di frontend serta Go pada AWS Lambda container image di backend.
- Mengimplementasikan autentikasi Amazon Cognito dengan role `reporter`, `agent`, dan `admin`, serta menerapkan RBAC di frontend dan backend.
- Membangun workflow tiket end-to-end mencakup pembuatan tiket, pencarian/filter/pagination server-side, assignment operator, komentar, lampiran S3 privat, dan audit trail aktivitas.
- Menyusun dokumentasi proyek, operator guide, dan API docs berbasis OpenAPI untuk mendukung deployment, demo, dan handoff teknis.

## Versi Teknis yang Lebih Kuat

- Built and deployed a cloud-native internal helpdesk application using React, Vite, TypeScript, Go, AWS Lambda container image, API Gateway HTTP API, DynamoDB, Cognito, S3, and AWS SAM.
- Implemented Cognito-based authentication and group-driven RBAC across frontend and backend for `reporter`, `agent`, and `admin` roles.
- Designed ticket lifecycle workflows with server-side filtering, sorting, pagination, operator assignment, append-only activity logs, secure file attachments via presigned S3 URLs, and OpenAPI-backed API documentation.
- Structured the project as a production-oriented full-stack system with live deployment, operator documentation, and presentation-ready technical packaging.
